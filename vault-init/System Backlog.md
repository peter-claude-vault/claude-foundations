# System Backlog

Append rows under the cluster H2 that fits the project's primary axis. Cluster H2 list is manifest-driven (`backlog.clusters[]`). Rename, merge, or split clusters by editing the librarian manifest — `backlog-hygiene` will route archive moves to whatever H2s the manifest declares.

Row schema: `Status | Category | Type | Location | Dependencies | Last Updated | Notes`. Status enum: `triaged`, `researching`, `briefed`, `planned`, `in-progress`, `blocked`, `complete`, `archived`. Long-form session history goes in the satellite at `Logs/backlog-progress/<slug>.md`; the Notes cell carries the wikilink pointer plus a one-line current-state summary.
